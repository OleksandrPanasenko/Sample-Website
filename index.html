UI Controls Demo with Explanations

1. Radio Buttons

Radio buttons дозволяє один (і тільки один) варіант із групи.

2. Checkboxes

Checkboxes дозволяють обрати одну, декілька, усі або жодну з опцій.

3. Text Inputs

Text inputs дозволяють вводити невелику кількість тексту (один рядок), до прикладу, в формах (email, пароль і т.д.).

4. Tabs

Tabs дозволяють мати доступ до різного контенту без залишання сторінки.

5. Buttons

Buttons дозволяють виконувати дії на сторінці, запускаючи onclick() подію та присутні в ній скрипти.

6. Text Labels

Labels надають опис для блоків тексту та інших елементів, автоматично переходять на новий рядок.

7. Links

Links дозволяють переходити на інші сторінки сайту, інші сайти та конкретні місця на сторінці. Текст описує посилання.

8. Tooltips

Tooltips показують додаткову інформацію, якщо навести на них курсором.

Hover over this text

9. Dropdown List

Dropdown lists дозволяють обрати один чи декілька елементів списку (може розгортатися при натисканні).

10. Data Grid (Table)

Data grids (tables) показує структуровану інформацію в табличному вигляді (тепер менше використовується).
